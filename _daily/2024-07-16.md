	# 父母 / 宗教
	皈依以来我生活中发生了很多正面改变，其中之一是和父母的关系。咨询师之前就提醒我，说切断关系也好、重建关系也好，无论怎样他都会支持我，但听起来我实际上还是希望和父母建立亲密关系的。我之前的纠结在于按照个人主义的文化立场，我完全无法说服自己去面对可能会令我痛苦的关系。但现在我的视角发生了很大改变，这不再是我的决定，而是主的指导。Abusive relationship是另一种情况，像我这种只是感觉没有得到过足够多、足够正面的爱，被这样推一把而去努力重建关系，我其实是受用的。
	
	抱着这样的心态很多事都看开一些。我会更主动和我妈视频，我妈看起来好开心，我其实也会因为自己能对另一个人产生这么大的影响而开心。晚上我边吃零食边聊天，我爸看到了来叨叨我说“你这等着长胖吧“。要是过去的我肯定会很不高兴，但我现在会意识到他主要是希望我健康，然后也不懂别的方式表达。我觉得因为对主的信任，我对父母的爱也有了更多信任。另一方面是我对身体和样貌少了很多执念，我更加追求纯净健康（比如通过纯净的饮食保持健康，朴素着装而避免杂念），而不是外在美，所以听到对我身体的评价不再有什么大的内心波动。某种意义上来说和自己的身体也有了更积极的关系。我是主所创造的生物，我的存在本身就是美的，不需要额外装饰。
	
	总体来说觉得自己心态平和很多，心中也充满包容，尤其是对自己的善念。这一切都让我充满感激。

	# 父母 / 心理学
	按科学/心理学的角度来说，这种心态改变其实就是motivated reasoning吧。过去是犹豫不决要不要重建关系，而同时因为不能与父母亲密而内心愧疚。为了自洽，在我的陈述中对父母的评价非常严厉，有选择地更关注负面的部分，因为同时考虑负面正面实在是太复杂了。就像我之前和咨询师说的，有时我宁愿父母有肢体虐待我，这反而能让我下定决心不再联系。正是因为他们是有局限的人但不是恶人，我的情绪混杂了失望和希望，让我不知所措。
	
	但现在是否重建关系不再是一个问题，而是主对我的期待。所以为了让关系能顺利重建，我会有选择地更关注好的那部分，也会给父母更多信任，把他们的行为解释为关照。而因为我对父母的态度更友善，他们也对我更友善、态度更积极。这让我看到更多开放沟通的希望，让我有勇气展现脆弱，逐渐产生正面循环。
